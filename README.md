# 🎮 Jogo da Velha

Projeto simples de **Jogo da Velha** desenvolvido utilizando **HTML, CSS e JavaScript**, com foco em praticar lógica de programação, manipulação do DOM e estilização de interfaces web.

## 📌 Sobre o Projeto

Este projeto recria o clássico jogo da velha diretamente no navegador, permitindo que dois jogadores joguem alternadamente.

O objetivo principal foi colocar em prática conceitos fundamentais do desenvolvimento web, como:

* Estruturação com HTML
* Estilização com CSS
* Manipulação de elementos com JavaScript
* Eventos de clique
* Lógica de vitória e empate

---

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

---

## 🎯 Funcionalidades

* ✅ Tabuleiro interativo
* ✅ Alternância automática entre jogadores
* ✅ Verificação de vitória
* ✅ Verificação de empate
* ✅ Reinício da partida
* ✅ Interface simples e responsiva

## ▶️ Como Executar

1. Clone este repositório:

```bash
git clone https://github.com/PasteldeSazon/jogo-da-velha.git
```

2. Acesse a pasta do projeto:

```bash
cd jogo-da-velha
```

3. Abra o arquivo `index.html` no navegador.
